# CoinGecko Crypto Market Report

Automated crypto market analysis deployed to GitHub Pages.

## 🌐 Live Report
**https://bruce-cham.github.io/CoinGecko/**

## 📊 What's Included
- **Market Structure**: Total cap, BTC/ETH dominance, volume
- **Momentum & Breadth**: Gainers/losers analysis
- **Top 10 Gainers/Losers**: With price and change %
- **Sector Rotation**: Top 10 crypto sectors
- **Trending Coins**: Real-time trending from CoinGecko

## ⏰ Update Schedule
- **8:00 AM GMT+8** (Morning report)
- **8:00 PM GMT+8** (Evening report)

## 🚀 How It Works
1. GitHub Actions runs twice daily
2. Fetches data from CoinGecko API
3. Generates HTML report
4. Deploys to GitHub Pages

## 🛠️ Local Development
```bash
pip install requests
python3 generate_html_report.py
# Open index.html in browser
```
